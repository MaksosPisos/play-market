# Play Market

Верстка главной страницы на React + TypeScript (Vite).

## Запуск

```bash
npm install
npm run dev
```

## Структура

- `src/components/` — UI-компоненты (Header, Hero, TopUp, Products, Reviews, Footer)
- `src/data/` — моковые данные
- `src/types/` — TypeScript-типы
- `public/assets/` — статика (иконки, картинки)
